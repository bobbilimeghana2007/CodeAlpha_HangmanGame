# Hangman Game - CodeAlpha Internship Task

## 📌 Project Description
This is a simple text-based Hangman Game developed using Python.

The player tries to guess a hidden word one letter at a time. The player has only 6 incorrect attempts.

## 🚀 Features
- Random word selection
- Tracks guessed letters
- Input validation
- Displays remaining attempts
- Game win/lose conditions

## 🛠 Technologies Used
- Python
- Random Module

## ▶ How to Run
1. Open terminal
2. Run:
python hangman.py
